# 吐槽

> 本文严谨转载 🚫🚫🚫

## 背景

曾经我参与到一个非常大的项目，当时去了很多前端同事都没法解决，领导很不满，后来就派我去了；期间发生了一件事情，那边有一个后端A工和一个测试，当出现问题的时候，那个后端对测试说：“你帮我远程连好，数据库连好，我去看下。”；

当时负责这个大项目的大领导有一次开会问了我一句话：“是不是我们公司前端都这个样子？”；我随即脱口而出：“你看A工，其实这个和岗位没有关系”；那位领导也没有再说话了。

虽然是我脱口而出的，但我一直在想解决方案，从绩效，任职资格，以及新员工培训等方面入手；但还是没有想到好的方案解决，可能是应试教育的原因，也可能只是单纯人性格的问题；马督工说，是问题都要带上解决方案，否则那都是耍流氓；我时常逛论坛，那戾气重，我也想摆一次烂，把自己这些年的都和不满吐槽一下；

## 有关称呼

认识我的都知道，我的签名是让大家叫我霄工，而不是相工，这个也经常成为人力的一个梗（我这么写也是为了减少一些不必要的沟通成本）；这是一个很有特色的海康文化，其实在海康之前，我还年轻，当时我喜欢测试叫我xx（我姓名的首字母缩写），我来公司的时候也想着是否能推行一下花名的文化，这样更有亲切感。不过我们部们经常叫我霄哥，我也习惯听霄工，感觉也很亲切。

至于公司有一些部门，动不动就称某某老师，某某大佬的，一次两次还好，听多了是真难受；还有一些人老是把我的`霄`打成`宵`的，虽然我没有说什么，但你问我问题我肯定是阴阳怪气的回你的。

开头说的大领导，她有一个非常好的习惯，就是第一次记名字的时候，她第一次接触一个新的同事在电脑上都会把拼音打正确，以确保后面每次打字的时候都会把名字输入正确；

## 有关UI交互

我刚来公司的时候，当时不是特别懂，我称我们UI交互，为美工；好像当时他们都不是很爽，一定要叫设计师；现在好像都叫UI交互设计师了；或者简称UI交互；可能以为美工就是个画页面的；

即使我现在查了下美工的定义：美工一般是指对平面，色彩 ，基调，创意等进行加工和创作的技术人才，分为平面美工、网页美工和三维美工。一般需要精通Photoshop等设计软件。

在做 hui 之前，大概17年的时候，当时做软件的人还少，我给当时的UI部门做过一次培训。讲的是 `css` 和 `html`；其实就是想让他们学习一下 `css`；他们也不是很乐意，现在很多设计师和我聊的很多也是设计理念，设计模式，设计规范，设计原则；而不会有人和我说，你这个`padding`写错了，不是`5px`应该是`2px`，设计师们你们可以尝试用 `html + css` 还原一下你们的设计稿，我觉得海康能做到这个的设计师不超过设计师 5%；我见过一个我认为海康最好的设计师，他有一次和我说，前端真有意思，我真的恨不得自己写，能不能教教我。

我一直认为设计师就是个虚的职业，要做好设计师，首先你得是美工，插画师，或者会更多工具的人；

## 有关测试

其实海康的测试我觉得悟性都还高的，和他们的职业有关系，他们会在整个软件流程中都出现，并担任重要的角色。而且现在各个部门的测试工具，也都是测试工程师自己写的，很多测试会找我学习前端工程。（其实我很想帮你们写，无奈部门不同）我认为最好的测试，会找到具体哪个接口报错，哪个页面的哪个样式有问题，甚至会看数据库表等等，其实我在海康还是遇到过一些，所以并不想吐槽测试们。

## 有关产品和管理

我并不太想说这2个岗位的人，主要是因为这个岗位其实级别更高，但影响的因素会有很多，也不是很好评价。但我想对其他岗位说一下，请站在他们的角度去思考问题，如果觉得对的那就要往上去改变产品和管理，而不是他们说什么你们做什么，不要一味的妥协。

## 有关公司

### 比较好的

工具的话，其实 `Hiklink` 挺好的，算是比较拿得出手的产品了，我可是经历过，skype 还有之前那个聊天软件，我名字都想不起来了。其他还有海康云，或者说是叫中心云计算平台，毕竟我们的 easy-serverless 就在这个上面；还有一些还行的，hikke，羽点，U+。

我觉得在海康其实只要你有想法和能力还是可以做很多事情的，公司还是有很多可以利用的资源，比如自己的云，自己的 npm 仓库等等，只要你有想法，把资源利用起来，你就可以做出很多东西，甚至你可以从 0 搭建一些东西，比如 iris；

### 有关文化

其实大家爱说 `体面`，出自一个事件，具体是什么我也不想回顾，有兴趣的可以去论坛搜下；但我觉得大家喜欢把 `体面` 当做一个贬义词来处理；其实我个人理解它是一个中性偏褒义的词语，代表的是一种隐忍；其实最合适用体面的，就是英国的绅士；结合我们公司的经营理念，代表着 `厚实`，厚积薄发；

> 我想问下，公司的经营理念（6字）和核心价值观（16字）应该没有人能够说得出来；不知道的可以去[海康威视官网](https://www.hikvision.com/cn/)的`关于我们`去看下；

这是我的个人理解，您也可以不同意；就像大家在说 `工具人`，理解仅仅是螺丝钉，只要负责干活就好了，但我认为也可以是用好工具、做好工具的人；

### 有关论坛与其他

其实中文互联网缺少高质量的讨论社区，这件事在海康也同样存在，论坛，HiKnow，wiki，其实都看不到优质内容以及优质评价。这个其实对大语言模型训练来说非常不利，有些优质的技术讨论圈子我也只能拉一些 hiklink 的讨论群。其实我觉得论坛有能力做这个事情，论坛的流量还是蛮大的，但每天我在上面看，戾气都太重。

> 说实话真的是羡慕 Github 的生态圈

当然不仅仅是论坛，其实我觉得国内社交平台也是一样的，比如微博，知乎，微信群，其实都是一样的；感觉现在的年轻人爱抱怨，特别的浮躁。我个人会在知乎关注一些海康威视的问题，会有一些入职前的同事来咨询问题，我也会给他们解答。其实论坛也是一样，也有零星的好技术讨论帖，我也会回一下。

总之大家还是多记录一些文献，多分享一些技术，多交流一些经验，这样才能让我们海康，或者我们国家的中文社区变得更好吧。也不仅仅为了大语言模型。

我比较喜欢一句话：**生命的遗产是对知识的更新**
