# 2025 我个人学习资源

之前的文献资源一般都是视频up主，但感觉一些学习资源不局限于视频；所以我就整理了一些学习资源，主要还是一些up主和一些博主相关的。

## 视频类 B 站 up 主

我个人包月充电的有：**秋芝 2046**，**Koala 聊开源**，**小黛晨读**，**Web3天空之城**

### 技术新闻类

+ **秋芝 2046** ⭐⭐⭐：AI 博主，分享的技术会偏应用和实战，通俗易懂；更新时间不定
+ **Koala 聊开源** ⭐⭐⭐：周更，分享每周开源项目（AI，前端，后端等）相关的新闻；会有一些技术类分享（会收费）
+ **胡泊Hubo** ⭐：一个设计师，主要关注 AI 相关的新闻
+ ai信息差 ⭐：这个我刚关注，如题，谨慎关注，更新太多

### 前端类

+ **oooooohmygosh** ⭐⭐：一个平面设计师，视频以分析设计理念为主（同事刚推荐，我也刚关注）
+ **zhangxinxu** ⭐⭐: 主要分享前端 css技术
+ **Simon_HeQAQ** ⭐：前端up主，主要做一些提效类工具
+ **远方os** ⭐：vue 官方成员，会分享一些 vue 的优秀实践，分享通俗易懂

### 新闻类

+ **小黛晨读** ⭐⭐⭐：都是一些实时的新闻，一般一周2更，我个人主要的新闻来源
+ **马督工** ⭐⭐：会针对一个社会新闻进行解读
+ **产经破壁机** ⭐⭐⭐：腾讯和睡前消息合作的社会现象解读，非常适合年轻人看~
+ **九妹瓜田** ⭐⭐： 社会热门事件列表

### 其他

+ **Web3天空之城** ⭐⭐：分享内容很杂，国外新闻偏多，也有一些有名的演讲，和国外优秀视频搬运，练听力挺好的，谨慎关注，更新太多
+ **罗翔说刑法** ⭐⭐⭐：法律科普，更多的他热爱看书
+ **火山哥哥** ⭐⭐：做小游戏的up主，非常有意思，会把做游戏的步骤一步一步拆解出来
+ **李永乐老师官方** ⭐⭐：一位数学老师，会分享一些数学知识解决生活问题
+ **YJango** ⭐⭐：教你如何学会学习的一个大学老师
+ **稚晖君** ⭐⭐⭐：中国"钢铁侠"，如今已经创业，视频发的少了，可以多翻翻他以前的视频看看
+ **代码之外BeyondCode** ⭐：分享技术人的生活
+ **老高与小茉** ⭐：油管up主，国外社会现象解读
