# 如何快速结缺陷单

## 背景

最近合规性的缺陷特别多，相信一些后端的同事都收到系统给大家提的缺陷，这些缺陷，有多有麻烦，还没有批量操作，很是讨厌；特别是我把缺陷都改完的那刻，还要花大量时间去结缺陷单的时候，气不大一处来。

当然缺陷最多的肯定是前端，其实这个我本身也找过做缺陷单的人。他们虽然有一些优化，但说实话力度不够。

还有就是一些同事也在做一些 e2e 的测试的预研，也在这里想介绍一个相对简单的方案

so，这里介绍一个奇技淫巧

> 以下技巧仅为了提高学习与工作效率，务必谨慎使用

## 篡改猴 (Tampermonkey) 

篡改猴 (Tampermonkey)，又称油猴，是拥有 超过 1000 万用户 的最流行的浏览器扩展之一。 它适用于 Chrome、Microsoft Edge、Safari、Opera Next 和 Firefox。

它允许用户自定义并增强您最喜爱的网页的功能。用户脚本是小型 JavaScript 程序，可用于向网页添加新功能或修改现有功能。使用 篡改猴，您可以轻松在任何网站上创建、管理和运行这些用户脚本。

简单来说他可以帮你执行输入页面 url 后想要做的动作。

## 其他用途

### e2e 测试

> e2e 测试，简单理解就是测试手动点浏览器上的功能

其实如果想写e2e测试，其实不需要用到什么 playwright 等一些专业工具，直接自己手写脚本就好了。这个在测试组很难落地，因为前端的元素经常会变

但后续互联网项目也回归到我们部门，互联网的简单灰度测试其实蛮值得使用一下的。

### 美化页面

如果对某个页面觉得太亮，可以直接装个插件，写个脚本就能搞定

现场客户觉得页面上要换个图标，不用修改代码，直接装个插件写个脚本~

### 其他油猴脚本

github 有很多油猴脚本，大家可以去自行搜索，比如可以用来抢票，去除广告，美化页面等
